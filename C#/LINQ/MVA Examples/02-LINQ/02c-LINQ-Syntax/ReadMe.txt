LINQ Syntax ReadMe

This demo explores various LINQ operators using lambda expressions. We then take a look
at query syntax rules and combining keywords with operators. 

Part A: LINQ Operators

1. Write a query to get customers in Oakland, sorted by last name
   - Use LINQ operators
   - Write same query using LINQ keywords

2. Get unique list of cities, sorted
   - Operators
   - Keywords with Distinct

3. Get customers in cities matching array of strings
   - Keywords with Contains

4. See if any customers have more than 100,000 orders
   - Any operator with a predicate

5. Get first three customers
   - Take operator

6. Reverse previous query

7. Create an empty sequence of customers
   - Print the Count
